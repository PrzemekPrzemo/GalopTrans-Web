# Hero banner assets

Place the bilingual hero banner images here. The page swaps between them
based on the selected language.

Expected files (referenced from `index.html`):

- `hero-pl.png` - Polish version of the banner
- `hero-en.png` - English version of the banner

If you prefer a different format (`.jpg`, `.svg`, `.webp`), update the
`src` / `data-src-pl` / `data-src-en` attributes on `#hero-banner` in
`index.html` to match.

Recommended export: ~2000 px wide, transparent or matching the page
background (`#fbf7ea`).
